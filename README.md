## 🏗️ Proyecto destacado — Citrix Control DAM

<div align="center">

```
┌─────────────────────────────────────────────────────────────┐
│                   CITRIX CONTROL DAM                        │
│         Plataforma de monitorización empresarial            │
├──────────────────────┬──────────────────────────────────────┤
│   Backend            │   Frontend / Mobile                  │
│                      │                                      │
│   Java 17            │   CitrixGO App                       │
│   Spring Boot 3      │   Svelte + TypeScript                │
│   REST API           │   Tailwind CSS                       │
│   WebSocket          │   Capacitor                          │
│   PowerShell SDK     │   Android · iOS · Web                │
│   Swing + JFreeChart │                                      │
└──────────────────────┴──────────────────────────────────────┘
```

</div>

**¿Qué hace?**
- 🖥️ **Monitorización en tiempo real** de entornos Citrix: usuarios, VDAs, grupos de entrega y aplicaciones
- ⚡ **WebSocket** para datos en vivo sin necesidad de recargar
- 🔀 **Ejecución multihilo** de scripts PowerShell para recopilar información del sistema
- 📊 **Gráficos interactivos** en interfaz de escritorio (JFreeChart)
- 📱 **App multiplataforma** (web, Android, iOS) con CitrixGO
- 🏛️ **Arquitectura escalable** con separación clara de capas
